# INGRID DMS User Manuals

This folder contains end-user documentation for the INGRID DMS MVP.

## Available Manuals

1. [Service / Fixed Ops User Manual](./SERVICE_FIXED_OPS_USER_MANUAL.md)
2. [INGRID DMS Object Map](./INGRID_DMS_OBJECT_MAP.md)

## Recommended Reading Order

For dealership users:

1. start with the service/fixed-ops manual
2. use the object map only if you need deeper workflow context

For internal product and implementation teams:

1. read the object map
2. then read the user manual to confirm the workflow is understandable from the desk level
